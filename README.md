Student Marks Management System

 Project Description

This is a C-based Student Marks Management System that allows users to manage student records efficiently through a simple command-line interface.

The system provides functionalities to add student records, display student details, and generate rank lists based on total marks. It simulates a basic academic record management system using C programming concepts such as structures, arrays, loops, and sorting.

Features

Add student records
Display all student records
Generate rank list
Automatic total mark calculation
Simple menu-driven interface
Easy data management

Technologies Used

C Programming Language

Menu Options

Check/Add Student Record
Display Students
Generate Rank List
Exit

 How to Run

Make sure a C compiler is installed on your system.

Download or copy the source code file.

Open terminal or command prompt in the project folder.

Compile the program using:

gcc student.c -o student

Run the program using:

./student
